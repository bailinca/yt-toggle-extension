# Toggle between YouTube and YouTube Music for the current video

## Load the extension in Chrome:

1. Go to `chrome://extensions`
2. Enable Developer mode (top right)
3. Click Load unpacked
4. Select your `yt-toggle-extension` folder
5. Open a [Youtube](https://www.youtube.com/watch?v=dQw4w9WgXcQ) or [YouTube Music video](https://music.youtube.com/watch?v=dQw4w9WgXcQ)
